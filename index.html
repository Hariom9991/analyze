Project Overview

\n

\n This project demonstrates a robust data processing workflow, taking raw Excel data, transforming it using a Python script, and publishing the results automatically via GitHub Actions and GitHub Pages.\n

\n

\n The `execute.py` script processes `data.csv` (converted from `data.xlsx`) and generates `result.json`. The entire process, from linting to execution and deployment, is automated through a GitHub Actions workflow.\n

\n\n

Key Features:

\n
    \n
  • Automated data conversion from Excel to CSV.
    • \n
  • Python script (`execute.py`) for data transformation and JSON output.
    • \n
  • Continuous Integration (CI) with Ruff for code quality.
    • \n
  • Automated generation and publication of `result.json` to GitHub Pages.
    • \n
  • Responsive UI built with Tailwind CSS.
    • \n
